# SPRINGLY
A demo Spring Boot application with authentication.

## Technologies

- Java 17
- Maven 4
- Spring Boot 3.3.1
- JSON Web Token 0.11.5
- PostgreSQL 17

## Dependencies

- Spring Web
- Spring Boot DevTools
- Spring Security
- Spring Data JPA
- Lombok
- JDBC API
- CycloneDX SBOM Support
- Jackson JWT
